PySeismoSoil Documentation
==========================

Hello, welcome to the documentation of PySeismoSoil.

PySeismoSoil is a Python library for performing 1D seismic site response analysis. The majority of the models in thie library are developed by the GeoQuake Research Group of the California Insitute of Technology.


Installation
------------

.. code-block:: bash

    pip install git+https://github.com/jsh9/PySeismoSoil

    
Dependencies
------------

* Python: 3.5+
* matplotlib: 2.0.0+
* numpy: 1.11.0+
* scipy: 1.1.0+
* numba: 0.38.0+
